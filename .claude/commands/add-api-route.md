Create a new FastAPI route endpoint following the backend-api skill patterns:

1. Create Pydantic models in `backend/app/models/`
2. Implement service logic in `backend/app/services/`
3. Add route in `backend/app/api/v1/`
4. Register router in `backend/app/main.py`
5. Test endpoint at `/docs` interactive API

Use async/await, proper error handling, and dependency injection.
